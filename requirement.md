# テナント配置の最適化要件仕様書

## 1. 空間設定
*   対象とするエリア: [0, 0] から [10000, 10000] の広さの2次元空間。

## 2. 配置する要素（テナント）
*   **フードコート**: 
    *   中心座標の目安: 点 (2000, 2000) の付近に位置すること。
    *   目標面積: 50000 平米。(重要度: 高)
*   **アパレルショップ**:
    *   中心座標の目安: 点 (2500, 2200) の付近に位置すること。
    *   目標面積: 30000 平米。(重要度: 中)
*   **スーパーマーケット**:
    *   中心座標の目安: 点 (4000, 5000) の付近に位置すること。
    *   目標面積: 80000 平米。(重要度: 高)
*   **シネマコンプレックス**:
    *   中心座標の目安: 点 (4500, 4800) の付近に位置すること。
    *   目標面積: 120000 平米。(重要度: 高)

## 3. 守るべき幾何制約
*   すべての要素は、指定された中心座標の目安点を必ず内部に含まなければならない。（最優先）
*   すべての要素は、対象エリアの外側にはみ出してはならない。（最優先）
*   要素同士は、互いに重なり合ってはならない。（優先度: 高）
*   各要素の面積は、できる限り目標面積に近づけること。（優先度: 中）
